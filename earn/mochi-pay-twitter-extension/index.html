About the Bounty

Mochi is a platform that allows Discord server owners to integrate trading supporting tools which is called Mochi Pay. Mochi Pay now provides 3 main functions:

  • Payment without wallet-equipped: User can transfer token in Discord or from Discord to other social platforms (Telegram, Twitter, and Email).
  • Users can create pay link and pay me request.
  • Users can organize airdrop on Discord

Superteam is sponsoring a bounty for developers to use Mochi Pay’s API to build a Twitter extension that allow users to perform these action on Twitter:

  • Connect Twitter account
  • Connect to existing Metamask/Sol crypto wallet
  • Deposit and Withdraw token
  • Create Paylink and payme request
  • Transfer token

To get more ideas, you can read this article: What can you build with Mochi APIs.

🎯 _**Your Mission**: Develop a payment tool by Twitter extension._

Evaluation Criteria

  • General Criteria
    • The extension should be developed for Chromium-based browsers
    • Ease of use of the extension (ie should have a good UX)
    • README Inclusions
      • A comprehensive README, incl. a clear description of it works and how Mochi APIs are being used in the “Technologies Used” section of the README
      • Please mention the team members’ information (GitHub handle, email)
      • Please include a link to the working demo or a testing guide
      • Include 3-5 suggested use cases for the extension
  • Submission Requirements
    • Was your Github repo accessible by the reviewing team at the time of submission? Please make sure that the submitted Github link is accessible or shared with gm@mochi.gg
    • A comprehensive README with all the things mentioned above
    • Note that the winning projects will need to be open-sourced once chosen
    • Only submit what you have built specifically for this bounty
    • A link to the working demo or a testing guide
    • A link to install your tool so we can experience it as an end-user
      • In the case of a browser extension, since the publication process may take a long time, it is also acceptable if you provide a link to the crx file
    • Did you submit before the deadline? Note: there will be no deadline extensions.
